# HeterogeneousRRI - Core Test

Testing core heterogeneous RRI capability via multiple input data methods (i.e. clear, draw, dispatch) and faciliating resource rotation.Establish baseline prior to rotation and compare against baseline after rotation.Track and validate resource creation and destruction semantics consistency after rotation.

# SupportPhone

> [!Important]
> SupportPhone is deprecated in Windows 10, version 1709.

`SupportPhone` specifies the telephone number(s) to call for support for the OEM.

## Values

| Value                   | Description                                                                           |
|:------------------------|:--------------------------------------------------------------------------------------|
| *phone_number*          | **Deprecated in Windows 10, version 1709**. Specifies the support telephone number(s) for the OEM. *Phone_number* is a string with a maximum length of 256Â characters.                                                                          |

This string type supports empty elements.

## XML Example

The following XML shows how to set OEM information.

```xml
<OEMInformation>
   <SupportProvider>Contoso Support</SupportProvider>
   <SupportAppURL>contoso-contact-support</SupportAppURL>
   <SupportURL>http://www.fabrikam.com/support</SupportURL>
</OEMInformation>
```
